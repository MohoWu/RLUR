Training Set

 

As variables are created under the ‘Variable creation’ tab, they are automatically appended to the training set attributes table. Alternatively, you can upload a pre-made training set to bypass the GIS operations.

 

·         Load existing dataset: Select a csv file. This can be the output saved from a previous modelling exercise. Note that with this option, a training set preview map is not plotted

 

·         Training set map: The monitoring data set in the ‘variable creation’ tab is previewed here. The attribute plotted is that set as the response in the ‘model builder’ tab. Use the slider to change the symbol size on the map.

 

·         Outlier selection: Click on a row in the attribute table to highlight it. This row will now be automatically excluded from any model (any defined model in the model builder will be automatically updated to reflect this). To reinstate a row, click it again to remove the highlight. To reinstate all rows, click the ‘clear’ button next to the outlier selection point counter. You will use the outlier remove capabilities after starting to build you models.
